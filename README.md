# Uppercase Selection

**Автор**: Всеволод Кузьмин, группа M3108
**Название плагина**: Uppercase Selection  
**Описание**: Плагин для Visual Studio Code, который преобразует выделенный текст в ВЕРХНИЙ РЕГИСТР.

## Возможности
- Работает с любым выделенным текстом в любом файле (.txt, .py, .js и т.д.).
- Поддерживает **множественное выделение** (несколько блоков текста одновременно).
- Ничего не делает, если текст не выделен (безопасно).
- Вызывается через палитру команд: `Ctrl+Shift+P` → "Uppercase Selection".

## Пример
Было выделено: `hello world`  
Стало: `HELLO WORLD`

## Установка
1. Откройте папку с плагином в VS Code.
2. Нажмите F5 для запуска в режиме отладки.
3. В новом окне выделите текст и вызовите команду.
